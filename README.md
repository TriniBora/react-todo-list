# TODO LIST

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).


This activity puts in practice the developing of React components with its own state, updating the state via callback functions, and practicing the use of form elements in React.

### Functionality Requirements:

As a user I will be able to add new TODOs to the list.
As a user I will be able to remove items from the TODOs list.
After adding a TODO item, the input must be cleared out.
The application will prevent the addition of duplicated items.

### Technical Requirements:

Must be done using NPM to create the React project.
The shouldComponentUpdate lifecycle method must be used in order to prevent duplicates.
Must be done using multiple components, it is not allowed to have only one component for all the application.
Trim start and end blank spaces when adding TODO items.

## Installation

Once you have downloaded this repository you have to execute the following command in order to fetch all the dependencies of the `package.json` file. (installation from an existing project)

```
npm install
```

This project includes dependecies such as:
- React
- FlipMove
- FontAwesome

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
